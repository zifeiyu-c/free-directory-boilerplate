## Features

- Listings (Tools, Products)
- Item Detail Page
- Categories & Tags
- Authentication (GitHub and Google)
- Submission (built-in)
- Sanity Studio (built-in CMS)
- Blog (hidden by default)
- Documentation (hidden by default)
- Analytics (Umami & Google Analytics)
- SEO (Sitemap, Open Graph)
- Modern UI (Shadcn UI)
- Responsive Design
- Multi-language (English & Chinese)
- Multi-theme (Light & Dark)


## Tech Stack

- Next.js 14
- NextAuth
- Database (PostgreSQL)
- Tailwind CSS
- Shadcn UI
- Lucide Icons
- Contentlayer
- Sanity
- Vercel

## How to use

1. Clone the repository
2. Run `pnpm install`
3. Configure the `.env` file
4. Run `pnpm dev`
